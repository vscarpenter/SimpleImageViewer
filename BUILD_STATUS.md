2. In the Project Navigator, find the `Services/AIAnalysis` folder
3. Right-click on `AIAnalysis` and select "Add Files to 'StillView - Simple Image Viewer'..."
4. Navigate to `StillView - Simple Image Viewer/Services/AIAnalysis/`
